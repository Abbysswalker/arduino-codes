### Español
Este código de Arduino controla tres LEDs conectados a los pines digitales 13 (rojo), 12 (verde) y 11 (azul) usando comandos ingresados a través del monitor serial. En la función `setup()`, se configuran los pines de los LEDs como salidas y se inicia la comunicación serial. En el bucle principal, el código espera que el usuario ingrese un color a través del monitor serial y enciende el LED correspondiente al color ingresado. También se admiten combinaciones de colores como cian, magenta, amarillo y naranja, así como un comando "low" para apagar todos los LEDs.

### Inglés
This Arduino code controls three LEDs connected to digital pins 13 (red), 12 (green), and 11 (blue) using commands entered through the serial monitor. In the `setup()` function, the LED pins are set as outputs, and serial communication is initiated. In the main loop, the code waits for the user to enter a color through the serial monitor and lights up the corresponding LED. It also supports color combinations like cyan, magenta, yellow, and orange, as well as a "low" command to turn off all LEDs.
