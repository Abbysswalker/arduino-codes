Este código de Arduino lee el valor de un sensor de luz conectado al pin analógico A5. Dependiendo de la cantidad de luz detectada, enciende un LED rojo conectado al pin digital 8 o un LED verde conectado al pin digital 9. Si el valor leído es menor de 200, se enciende el LED rojo y se apaga el LED verde. Si el valor es mayor o igual a 200, se enciende el LED verde y se apaga el LED rojo. El valor del sensor de luz se imprime en la consola serial con un retraso de 100 milisegundos entre lecturas.

Inglés
This Arduino code reads the value from a light sensor connected to analog pin A5. Depending on the amount of light detected, it turns on a red LED connected to digital pin 8 or a green LED connected to digital pin 9. If the read value is less than 200, the red LED is turned on and the green LED is turned off. If the value is 200 or greater, the green LED is turned on and the red LED is turned off. The light sensor value is printed to the serial console with a 100-millisecond delay between readings.
