### Español
Este código de Arduino controla un LED conectado al pin digital 7 y un botón conectado al pin analógico A0. Cuando el botón se presiona (detecta una transición de alto a bajo), el estado del LED cambia: si el LED está apagado, se enciende; si está encendido, se apaga. El código también incluye un pequeño retraso (`dt`) para evitar rebotes del botón.

### Inglés
This Arduino code controls an LED connected to digital pin 7 and a button connected to analog pin A0. When the button is pressed (detects a transition from high to low), the state of the LED toggles: if the LED is off, it turns on; if it is on, it turns off. The code also includes a small delay (`dt`) to debounce the button.
