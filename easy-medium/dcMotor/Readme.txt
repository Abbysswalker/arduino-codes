### Español
Este código de Arduino controla un motor utilizando un controlador de motor. Los pines digitales 11, 12 y 13 se utilizan para el control del motor: el pin 11 ajusta la velocidad del motor mediante PWM (modulación por ancho de pulso), mientras que los pines 12 y 13 controlan la dirección del motor. En la función `setup()`, se configuran estos pines como salidas y se inicia la comunicación serial. En el bucle principal, el código establece la dirección del motor y luego ajusta la velocidad a su valor máximo (`255`).

### Inglés
This Arduino code controls a motor using a motor driver. Digital pins 11, 12, and 13 are used for motor control: pin 11 adjusts the motor speed via PWM (Pulse Width Modulation), while pins 12 and 13 control the motor direction. In the `setup()` function, these pins are set as outputs and serial communication is initiated. In the main loop, the code sets the motor direction and then adjusts the speed to its maximum value (`255`).
