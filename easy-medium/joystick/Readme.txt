### Español
Este código de Arduino lee el estado de un botón y dos pines analógicos de un joystick. El botón está conectado al pin digital 7 y los pines analógicos del joystick están conectados a A0 (eje X) y A1 (eje Y). En la función `setup()`, se configuran los pines del joystick como entradas y el pin del zumbador como salida. La línea `digitalWrite(buttonPin, HIGH)` se utiliza para activar la resistencia de pull-up en el pin del botón, lo que se puede resumir con `pinMode(buttonPin, INPUT_PULLUP)`. En el bucle principal, el código lee los valores del joystick y del botón, y los imprime en la consola serial. Si se presiona el botón del joystick, el zumbador se activa; de lo contrario, permanece apagado. También incluye un pequeño retraso de 100 milisegundos (`dt`) entre lecturas para suavizar la salida.

### Inglés
This Arduino code reads the state of a button and two analog pins from a joystick. The button is connected to digital pin 7, and the joystick's analog pins are connected to A0 (X-axis) and A1 (Y-axis). In the `setup()` function, the joystick pins are set as inputs, and the buzzer pin is set as an output. The line `digitalWrite(buttonPin, HIGH)` is used to activate the pull-up resistor on the button pin, which can be summarized with `pinMode(buttonPin, INPUT_PULLUP)`. In the main loop, the code reads the joystick values and the button, and prints them to the serial console. If the joystick button is pressed, the buzzer is activated; otherwise, it remains off. It also includes a small delay of 100 milliseconds (`dt`) between readings to smooth the output.
