### Español
Este código de Arduino controla un motor paso a paso utilizando la biblioteca `Stepper`. El motor está configurado para 2048 pasos por revolución y una velocidad de 10 RPM. En la función `setup()`, se establece la velocidad del motor. En el bucle principal, el motor gira una revolución completa en una dirección, luego hace una pausa de 500 milisegundos (`dt`), y después gira una revolución completa en la dirección opuesta, seguido de otra pausa de 500 milisegundos. Este ciclo se repite indefinidamente.

### Inglés
This Arduino code controls a stepper motor using the `Stepper` library. The motor is set to 2048 steps per revolution and a speed of 10 RPM. In the `setup()` function, the motor speed is set. In the main loop, the motor turns one full revolution in one direction, then pauses for 500 milliseconds (`dt`), and then turns one full revolution in the opposite direction, followed by another 500-millisecond pause. This cycle repeats indefinitely.
