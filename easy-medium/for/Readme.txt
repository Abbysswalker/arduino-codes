### Español
Este código de Arduino controla dos LEDs: uno rojo conectado al pin digital 4 y uno amarillo conectado al pin digital 3. En la función de configuración, ambos pines se configuran como salidas. En el bucle principal, el LED amarillo parpadea tres veces y luego el LED rojo parpadea cinco veces. Cada parpadeo tiene una duración definida por `dt`, que es de 500 milisegundos. El código comentado muestra una versión alternativa donde los LEDs parpadean sin usar bucles `for`.

### Inglés
This Arduino code controls two LEDs: a red LED connected to digital pin 4 and a yellow LED connected to digital pin 3. In the setup function, both pins are set as outputs. In the main loop, the yellow LED blinks three times and then the red LED blinks five times. Each blink duration is defined by `dt`, which is 500 milliseconds. The commented-out code shows an alternative version where the LEDs blink without using `for` loops.
