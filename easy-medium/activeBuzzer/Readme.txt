### Español
Este código de Arduino controla un zumbador conectado al pin digital 8. En la configuración, el pin del zumbador se define como salida. En el bucle principal, el zumbador emite pitidos a intervalos de tiempo definidos por `dt`, con un retardo de 1000 microsegundos entre encendido y apagado. El código comentado sugiere una funcionalidad adicional para que el usuario ingrese un número de pitidos a través del monitor serial, pero esta parte no se ejecuta en el código actual.

### Inglés
This Arduino code controls a buzzer connected to digital pin 8. In the setup, the buzzer pin is defined as an output. In the main loop, the buzzer emits beeps at time intervals defined by `dt`, with a delay of 1000 microseconds between on and off states. The commented-out code suggests an additional functionality for the user to input a number of beeps via the serial monitor, but this part is not executed in the current code.
