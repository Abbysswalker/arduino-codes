### Español
Este código de Arduino controla tres LEDs conectados a los pines digitales 9 (rojo), 10 (amarillo) y 11 (azul) usando comandos ingresados a través del monitor serial. En la función `setup()`, se configuran los pines de los LEDs como salidas y se inicia la comunicación serial. En el bucle principal, el código espera que el usuario ingrese un color (rojo, amarillo o azul) a través del monitor serial y enciende el LED correspondiente al color ingresado. El código también contiene una sección comentada para saludar al usuario por su nombre, pero esta parte no se ejecuta en el código actual.

### Inglés
This Arduino code controls three LEDs connected to digital pins 9 (red), 10 (yellow), and 11 (blue) using commands entered through the serial monitor. In the `setup()` function, the LED pins are set as outputs and serial communication is initiated. In the main loop, the code waits for the user to enter a color (red, yellow, or blue) through the serial monitor and lights up the corresponding LED. The code also contains a commented-out section intended to greet the user by their name, but this part is not executed in the current code.
