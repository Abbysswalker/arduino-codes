### Español
Este código de Arduino lee el estado de un botón conectado al pin digital 4 y controla un LED conectado al pin digital 2. Cuando el botón está presionado, el LED se enciende; cuando el botón no está presionado, el LED se apaga. Además, el valor leído del botón se envía a la consola serial para su monitoreo.

### Inglés
This Arduino code reads the state of a button connected to digital pin 4 and controls an LED connected to digital pin 2. When the button is pressed, the LED turns on; when the button is not pressed, the LED turns off. Additionally, the value read from the button is sent to the serial console for monitoring.
