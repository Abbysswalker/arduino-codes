### Español
Este código de Arduino controla un LED conectado al pin digital 2 utilizando un potenciómetro conectado al pin analógico A1. En la función `setup()`, se configuran los pines del potenciómetro como entrada y el del LED como salida, y se inicia la comunicación serial. En el bucle principal, el código lee el valor del potenciómetro y lo imprime en la consola serial. Si el valor del potenciómetro es mayor a 1000, el LED se enciende y permanece encendido mientras el valor sea mayor a 1000, actualizando el valor leído del potenciómetro en cada iteración. Si el valor es menor o igual a 1000, el LED se apaga. El código también incluye un retraso de 400 milisegundos (`dt`) entre lecturas para suavizar la salida.

### Inglés
This Arduino code controls an LED connected to digital pin 2 using a potentiometer connected to analog pin A1. In the `setup()` function, the potentiometer pin is set as an input and the LED pin as an output, and serial communication is initiated. In the main loop, the code reads the potentiometer value and prints it to the serial console. If the potentiometer value is greater than 1000, the LED turns on and stays on as long as the value is greater than 1000, updating the potentiometer value in each iteration. If the value is less than or equal to 1000, the LED turns off. The code also includes a delay of 400 milliseconds (`dt`) between readings to smooth the output.
