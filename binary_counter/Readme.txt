### Español
Este código de Arduino es un contador binario que controla cuatro pines digitales (pin1 a pin4). Cada pin representa un bit de un número binario. El programa enciende y apaga los pines en una secuencia que cuenta de 0 a 15 en binario, con un retardo de 500 milisegundos entre cada cambio de estado. Es ideal para demostrar cómo funcionan los contadores binarios utilizando LEDs.

### Inglés
This Arduino code is a binary counter that controls four digital pins (pin1 to pin4). Each pin represents a bit of a binary number. The program turns the pins on and off in a sequence that counts from 0 to 15 in binary, with a delay of 500 milliseconds between each state change. It is ideal for demonstrating how binary counters work using LEDs.
